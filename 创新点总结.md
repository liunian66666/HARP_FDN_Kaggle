# HARP-FDN-PCX 创新点总结

## 模型名称

**HARP-FDN-PCX: Horizon-Aware Relative Prototype Forecasting Decomposition Network with Posterior Calibration**

该模型是在 InterFDN 的非均匀支撑点预测框架上进一步扩展得到的。核心思想不是简单增加分支，而是将原来的“支撑点分类期望预测”升级为“概率分布预测、相对原型解码、跨尺度分布一致性与后验不确定性校准”的统一框架。

## 总体框架

模型仍保留 InterFDN 的基础思想：

- 使用 RevIN 处理序列分布偏移；
- 使用 EMA/DEMA 等分解方式获得 seasonal 与 trend；
- 使用非均匀支撑点将连续预测转化为概率支撑点预测；
- 使用 fine/coarse 两个尺度约束未来序列结构。

在此基础上，HARP-FDN-PCX 对预测方式、尺度一致性和输出校准进行了结构性升级。

## 创新点一：Horizon-Aware Relative Prototype Decoding

InterFDN 原始做法是直接在非均匀支撑点上分类，并通过期望得到预测值。这种方式对短预测较有效，但在长预测时，直接预测绝对值容易产生累计偏移。

HARP-FDN-PCX 引入 **相对原型解码机制**：

1. 先由 coarse 分支预测未来整体原型，作为未来轨迹的 anchor；
2. fine 分支不再只预测绝对值，而是预测相对于该 anchor 的 residual support；
3. 通过 anchor certainty gate 控制模型在每个 horizon 和 channel 上使用绝对解码还是相对残差解码。

这种设计使模型从“直接预测未来点值”转变为“先估计未来整体原型，再预测局部偏移”，从而增强长预测场景下的稳定性。

### 可解释性

该模块具有明确的可解释含义：

- coarse output 表示未来趋势或形态原型；
- fine residual 表示局部修正；
- relative gate 表示模型对原型约束的依赖程度；
- anchor variance 表示原型预测的不确定性。

因此，该模块不仅提升预测能力，也能解释模型在不同预测步上是依赖整体趋势还是局部细节。

## 创新点二：Distribution-Aware Cross-Scale Consistency

InterFDN 的尺度一致性主要是点值一致性，即让 fine 结果下采样后接近 coarse 结果。该方式只约束预测均值，忽略了支撑点 posterior 中包含的不确定性信息。

HARP-FDN-PCX 将尺度一致性扩展为 **分布一致性约束**：

1. 从 support posterior 中同时解码预测均值和预测方差；
2. 将 fine 分支的预测分布聚合到 coarse 时间尺度；
3. 约束聚合后的 fine 分布与 coarse 分布在均值和方差上保持一致。

该设计使模型不仅要求不同尺度的预测值一致，还要求不同尺度对未来不确定性的估计一致。

### 贡献

该模块把 InterFDN 的支撑点概率输出从“只用于求期望”扩展为真正的分布建模工具。相比普通多尺度 MSE 约束，它更充分利用了 support posterior 的信息。

## 创新点三：Posterior-Gated Residual Calibration

HARP-FDN-PCX 引入一个弱校准分支，用于修正支撑点离散化带来的 quantization bias。

该校准不是无条件加入输出，而是由 posterior uncertainty 控制：

- 当模型预测方差较低、posterior 较确定时，校准分支几乎不影响输出；
- 当模型预测方差较高、posterior 较不确定时，允许校准分支进行轻微修正；
- 校准强度为可学习参数，并采用较小初始化，避免破坏主预测结构。

该模块的作用可以理解为：在模型不确定的位置，利用历史输入提供一个保守的 residual correction。

## 辅助结构：Probability Mixer Network

为了生成更稳定的 support logits，模型将原始预测网络替换为概率型 Patch Mixer 结构。

该网络包含：

- short-patch stream：捕捉局部变化和短期形变；
- long-patch stream：捕捉周期片段和较长模式；
- input-conditioned scale gate：根据输入自适应融合 short/long 表征；
- seasonal linear path：保留直接 seasonal 投影；
- trend gated path：对 trend 进行门控外推；
- support logit head：输出非均匀支撑点上的概率分布。

该结构服务于主创新点：它不直接输出预测值，而是输出支撑点分布，为相对原型解码、分布一致性和不确定性校准提供基础。

## 与 InterFDN 的区别

| 方面 | InterFDN | HARP-FDN-PCX |
|---|---|---|
| 解码方式 | 支撑点分类期望 | 绝对支撑点解码 + 相对原型残差解码 |
| 尺度一致性 | 点值 MSE 一致性 | 均值和方差的分布一致性 |
| 不确定性利用 | 主要用于分类置信度 | 用于分支融合、尺度一致性和后验校准 |
| 预测网络 | 原始 Network | 多尺度 Probability Mixer |
| 可解释性 | 支撑点概率和尺度输出 | 原型、残差、gate、variance 多层解释 |

## 可作为论文中的贡献表述

1. 提出一种 horizon-aware relative prototype decoding mechanism，将未来预测分解为 coarse prototype 和 fine residual correction，缓解长预测中直接绝对值解码带来的漂移问题。

2. 提出 distribution-aware cross-scale consistency，将 fine/coarse 尺度约束从点值一致性扩展到均值与方差的一致性，使非均匀支撑点 posterior 得到更充分利用。

3. 设计 posterior-gated residual calibration，在预测不确定区域自适应引入弱残差校准，以减小支撑点离散化误差并保持主预测结构稳定。

4. 构建 probability mixer support-logit generator，通过 short/long patch streams 和 input-conditioned gate 生成更稳定的支撑点概率分布。

## 不建议作为创新点表述的内容

以下内容更适合作为训练策略或实现细节，不建议写成模型创新点：

- `use_vali_ratio`：这是验证阶段 checkpoint 选择策略，不是模型结构；
- `batch_size`、`learning_rate`、`mse_loss_weight`：属于训练超参数；
- 单纯打开或关闭模块开关：不能单独构成创新点；
- 只描述“多尺度融合”：该说法过于常见，需要结合相对原型解码和分布一致性来讲。

## 推荐论文叙述主线

本文认为，基于非均匀支撑点的预测模型虽然能够缓解连续值回归的不稳定性，但原始支撑点期望解码仍存在两个问题：

1. 长预测中直接绝对值解码容易产生 horizon-wise drift；
2. 支撑点 posterior 中的不确定性信息没有被充分利用。

为此，本文提出 HARP-FDN-PCX。模型首先通过 coarse branch 建立未来轨迹原型，再通过 fine branch 预测相对于该原型的残差分布；同时，模型在 fine/coarse 尺度之间显式约束均值和方差一致性，并在高不确定区域引入 posterior-gated residual calibration。该框架使模型同时具备预测精度、跨尺度稳定性和更强的可解释性。
